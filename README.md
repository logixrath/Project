# Project

This report details the comprehensive kinematic analysis of two fundamental mechanisms in the study of machine dynamics: the slider-crank mechanism and the four-bar mechanism. The primary objective was to develop computational models capable of accurately determining the instantaneous velocities and accelerations of the constituent links across a full range of motion.
For the slider-crank mechanism, the analysis focuses on calculating the angular velocity (ω₂) and angular acceleration (α₂) of the connecting rod, and the linear velocity (Vₛ) and acceleration (aₛ) of the slider.
The four-bar mechanism analysis determines the angular velocities (ω₃, ω₄) and angular accelerations (α₃, α₄) of the coupler and follower links, respectively.
The implementation utilized expert-level kinematic equations, which were coded to allow for parametric analysis based on varied link lengths, input angular velocity, and crank angle.
